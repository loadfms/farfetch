# Farfetch

Tool to change themes on Linux Distros.

## Description

Change themes for your Alacritty, Polybar and Rofi easily.

## Getting Started

### Dependencies

* Alacritty
* Polybar
* Rofi

### Installing

* Clone de Repo
```
gh clone loadfms/farfetch
```

* Add your alacritty themes to you `$HOME/.alacrittythemes/`

### Executing program

* Build with
```
cargo build --release
```
* Execute
```
./target/release/farfetch
```
